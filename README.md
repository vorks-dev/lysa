# Lysa - Photo Organizer

**Lysa - Photo Organizer** is a lightweight desktop tool by **vorks. DEV** for browsing, viewing, and organizing photos stored in local directories. It automatically extracts file and EXIF metadata, enabling quick sorting and smooth navigation through your photo library.

## Features
- **Gallery Browsing**
  Load photos from a chosen root folder and navigate its subdirectories as albums.

- **Metadata-Powered Sorting**
  Sort your gallery by various file and EXIF attributes, extracted automatically.

- **Full-Resolution Viewer**
  Zoom, pan, and inspect detailed metadata while viewing photos in high resolution.

## Installation
Download the latest release from [GitHub Releases](https://github.com/vorks-dev/lysa/releases/latest).

## Usage
1. Select a root folder containing your photos.
2. Browse through automatically generated galleries and album subfolders.
3. Use sorting options to reorganize your gallery on the fly.
4. Open any photo to view it in full resolution with metadata.
